# Veriver NC est une application web et mobile pour la protection de l'environnement en Nouvelle-Calédonie.

---

## Fonctionnalités principales

Signalement de zones polluées avec photos, description et localisation
Carte interactive des signalements
Organisation et participation à des événements de nettoyage
Profils utilisateurs et suivi des contributions
Système de points, badges et classement
Notifications pour événements et mises à jour

---

## Stack technologique

Backend : Java 17+, Spring Boot 3, PostgreSQL, API REST, JWT
Frontend Web : Angular 17+, Tailwind CSS/Bootstrap, Leaflet ou Google Maps
Frontend Mobile : Flutter / React Native / Ionic

---

## Modules principaux

Configuration initiale (BDD, backend, frontend)
Base de données et entités
Backend : authentification, sécurité, gestion utilisateurs
Backend : gestion des signalements et événements
Backend : gamification (points, badges, classements)
Backend : notifications
Frontend Web : authentification, carte interactive, signalements, événements, profil, gamification, notifications, administration
Frontend Mobile : authentification, carte, signalements, événements, profil, notifications push, fonctionnalités avancées
Tests backend, frontend et mobile
Déploiement backend, frontend, mobile et infrastructure
Documentation technique et utilisateur
Fonctionnalités avancées et bonus (statistiques, intégration collectivités, IA)
Roadmap en 3 phases : MVP, mobile et gamification, fonctionnalités avancées

---

## Objectif

Créer une application complète pour impliquer les citoyens et les collectivités dans la protection de l'environnement en Nouvelle-Calédonie.
